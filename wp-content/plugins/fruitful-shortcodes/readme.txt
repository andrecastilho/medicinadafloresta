== Installation ==

1. Upload `fruitful-shortcodes` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Edit any Post or Page.
4. Click to the Fruitful Shortcodes button on the top of Content editor
5. Add shortcodes from popup.
